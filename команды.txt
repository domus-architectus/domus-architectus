1. Подготовка к «записи» (если изменил файлы)
После того как ты отредактировал файлы на компьютере, в терминале (в папке проекта) вводи по очереди:

git add . — говорит Git: «взять в работу все изменения во всех файлах».

git commit -m "Update: переход на списочный формат библиотеки" — фиксирует состояние проекта с коротким описанием.

2. Отправка на «стройку» (GitHub)
git push — отправляет все записанные изменения на GitHub.

Через 30 секунд Netlify подхватит их и обновит сайт.
